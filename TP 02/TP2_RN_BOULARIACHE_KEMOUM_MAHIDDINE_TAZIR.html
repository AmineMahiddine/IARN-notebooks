def computeCostNonVect(X, y, theta):
    m = len(y)
    sum = 0

def computeCost(X, y, theta):
    # calculer le cot AVEC vectorisation, 
    # comparer le temps de traitement
    
    m = y.shape[0] #the number of training examples
    return np.sum(np.power((np.dot(X, theta) - y), 2)) / (2 * m)
  • 2- Fonction de la descente du gradient

Cette fonction mettra jour les paramtres $\theta_0,\theta_1$ jusqu' convergence: atteinte du nombre d'itrations max, ou drive assez petite.

In [7]:
def gradientDescent(X, y, theta, alpha, iterations):
    
    m = y.shape[0] #len(X)                                                             
    all_cost = []
    all_theta = [theta]
    for i in range(iterations):                                                      
        all_cost.append(computeCost(X, y, all_theta[-1])) # garder aussi le cout  chaque itration 
        all_theta.append(all_theta[-1] + float(alpha) * (np.dot(X.T, (y - np.dot(X,all_theta[-1]))) /m)  )                     
    return all_theta,all_cost

Mise à l'échelle Et Normalisation des données en moyenne = 0 :¶

The data can be normalized by subtracting the mean () of each feature and a division by the standard deviation (). This way, each feature has a mean of 0 and a standard deviation of 1. This results in faster convergence.
